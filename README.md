This folder contains the data parsed from a case study involving interviews with five older adults, regarding their preferences of what tasks they want to delegate to robots across various household tasks.
Each file contains data obtained from one person. The columns **Action**, **Activity**, **Objects**, and **Locations**, define the task. The column **Preference** contains the preferred robot behavior. The column **Conditions** and **Conditional Preferences** are optional, and contain a condition under which a different preference applies. The **Explanation** and **Explanation Conditional** columns are optional and can contain the usersâ€™ explanations for all their preferences.
